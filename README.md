# 💸 Expense Tracker & Financial Insights Dashboard

## 📌 Overview

This project is an **industry-oriented Expense Tracker Dashboard** built using Data Science and Python.
It helps analyze personal spending patterns, detect overspending behavior, and generate actionable financial insights.

The project uses **synthetic data** to simulate real-world financial transactions and demonstrates end-to-end data analysis, visualization, and dashboard development.

---

## 🎯 Objectives

* Track and analyze daily expenses
* Understand category-wise spending behavior
* Identify overspending patterns
* Monitor savings trends
* Build an interactive dashboard for decision-making

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** → Data Processing
* **Matplotlib & Seaborn** → Static Visualization
* **Plotly** → Interactive Charts
* **Streamlit** → Web Dashboard

## ▶️ How to Run the Project

### 1. Clone Repository

```
git clone https://github.com/yourusername/expense-tracker-data-science.git
cd expense-tracker-data-science
```

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   (Windows)
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Generate Dataset

```
python src/data_generation.py
```

### 5. Run Dashboard

```
streamlit run app/streamlit_app.py
```
